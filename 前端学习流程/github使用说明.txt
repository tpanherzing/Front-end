github update后，增加了客户端的功能，意思是我们所有的工作都可以在客户端完成。然后传输到github.com上。)

前面都是俗语，后面是步zou.

1. 登陆github.com注册自己的用户名字
2. 创建一个新的仓库（new repository）->跳转到下一页面后一定选择public->在最下面选择初始化这个仓库带一个文件（initlalize this repository with a README(这里的README就是一个类似于html的标记语言写出的文本文件，目的是对于上面仓库中的代码做出一些解释)）
3.创建成功后，跳转到下一个界面，到了这个界面后，我要说明的是commit这个词语。当创建一个新的版本时，它上面就会出现标记。目前是第一版本，所以现在是数字1commit.
4.然后创建一个新的文件，creat a new file. 完成后会出现此时的版本变成了2commit.
